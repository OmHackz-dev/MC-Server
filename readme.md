# ᴍᴀɴɢᴏ ᴀɴᴀʀᴄʜʏ 🍋

## ᴀʙᴏᴜᴛ

**ᴍᴀɴɢᴏ ᴀɴᴀʀᴄʜʏ** ɪѕ ᴀ **ᴘᴜʀᴇ ᴍɪɴᴇᴄʀᴀꜰᴛ ᴀɴᴀʀᴄʜʏ ѕᴇʀᴠᴇʀ** ʜᴏѕᴛᴇᴅ ᴜѕɪɴɢ **ɢɪᴛʜᴜʙ ᴄᴏᴅᴇѕᴘᴀᴄᴇѕ**.
ɴᴏ ʀᴜʟᴇѕ. ɴᴏ ʀᴇѕᴇᴛѕ. ɴᴏ ᴀᴅᴍɪɴ ɪɴᴛᴇʀꜰᴇʀᴇɴᴄᴇ.
ᴊᴜѕᴛ ᴘᴜʀᴇ ѕᴜʀᴠɪᴠᴀʟ.

## ᴡᴀʀɴɪɴɢ
